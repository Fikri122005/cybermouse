# Sound Assets

Masukkan file audio berikut ke folder ini:

- `bgm.wav`
- `jump.wav`
- `coin.wav`
- `cheese.wav`
- `gameover.wav`
- `finish.wav`

Catatan:
- Gunakan format PCM WAV umum (16-bit, 44.1kHz) untuk kompatibilitas terbaik.
- Jika nama file berbeda, update path di `src/game/SoundManager.java`.
